# Senior-Discount-Checker
A straightforward Python script that evaluates a person’s eligibility for a senior citizen discount based on their age. This program exemplifies simple decision-making processes in code, applying a single condition (age 65 or older) to determine eligibility. It’s a practical demonstration of `if-else` statements and user interaction.
Key Features:

	•	Checks age against a predefined eligibility criterion (65+).
	•	Provides clear output indicating discount eligibility.
	•	Incorporates basic input validation for age.
	•	Suitable for understanding conditional logic and practical application of programming rules.
